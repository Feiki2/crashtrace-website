Alle Features

Was CrashTrace
kann.

CrashTrace liefert alles, was Ihr IT-Team braucht, um Abstürze lückenlos zu dokumentieren und schnell zu lösen.

📡

Echtzeit-Systemüberwachung

CPU, RAM, Disk, Temperaturen und Netzwerk werden sekündlich erfasst und in einem Rolling-Buffer gespeichert.

📸

Automatische Screenshots

Alle 5 Sekunden wird ein Screenshot aller Monitore gesichert. Exakt was der User zuletzt getan hat.

📧

Automatischer E-Mail-Report

Nach jedem Absturz: ZIP-Report per E-Mail an das IT-Team – vollautomatisch.

💥

BSOD-Erkennung

Zuverlässige Erkennung via Windows Sentinel + Kernel-Power Event ID 41.

🔒

Windows Event Log

Vollständige Integration. Alle relevanten Ereignisse werden erfasst.

🖥️

Admin-Oberfläche

Grafische Admin-GUI. Übersichtlich, intuitiv, professionell.

🌐

Netzwerkaktivität

Vollständige Protokollierung von Netzwerkverbindungen zum Crash-Zeitpunkt.

⚙️

Prozess-Inventar

Welche Prozesse liefen beim Absturz? Vollständige Liste, immer dabei.

🔧

Treiber-Informationen

Installierte Treiber und Versionen – häufig die versteckte Crash-Ursache.

Erkennungsmethoden

Wie CrashTrace erkennt.

🛡️

Windows Sentinel

Windows Sentinel Mechanismus für zuverlässige BSOD-Erkennung.

⚡

Kernel-Power Event ID 41

Erkennt auch unerwartete Neustarts durch Strom- oder Hardware-Fehler.

🔍

App-Crash-Erkennung

Anwendungsabstürze mit vollständigen Stack-Trace-Informationen.

📊

Rolling Buffer

Kontinuierlicher Buffer – die letzten 30–300 Sekunden immer verfügbar.
